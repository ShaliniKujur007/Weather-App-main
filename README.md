# 🌤️ Weather App

A modern and responsive weather application built using HTML, CSS, and JavaScript. The project focuses on creating a clean, interactive, and user-friendly interface while strengthening front-end web development skills.

---

# 📖 Introduction

The Weather App is a front-end web application designed to display weather information through an intuitive and responsive interface. It was developed as a learning project to practice building responsive web pages, handling user interactions, and organizing JavaScript code.

This project emphasizes simplicity, clean design, and usability while providing a solid foundation for future enhancements.

---

# ✨ Features

- 🌍 Search weather by city
- 📱 Responsive design
- 🎨 Clean and modern UI
- ⚡ Fast and lightweight
- 🌡 Weather information display
- 💧 Humidity section
- 💨 Wind speed section
- ☁ Weather condition display
- 🖥 Cross-browser compatible
- 🔍 Easy-to-use search functionality

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

# 🚀 Development Process

The project was developed following these steps:

### 1. Planning
- Defined the application layout.
- Identified the information to display.
- Created a simple wireframe for the interface.

### 2. Designing
- Built the user interface using HTML.
- Styled the application using CSS.
- Implemented a responsive layout for different screen sizes.

### 3. Development
- Added JavaScript for user interaction.
- Implemented search functionality.
- Organized the code into reusable sections.

### 4. Testing
- Tested responsiveness on different screen sizes.
- Verified UI consistency.
- Checked browser compatibility.

### 5. Refinement
- Improved spacing and typography.
- Optimized the layout.
- Enhanced the overall user experience.
